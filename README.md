# 使用说明
1. 进入结构解析目录,新建数据库连接。
2. 选择数据源后左下方会出现数据库中的表,填写包名前缀和删除前缀并且选中表后可以选择模板直接生成代码,也可以生成表定义进行配置后再生成代码
3. 进入结构定义管理,选择数据源后会出现已经导入表定义,点击编辑按钮会弹出编辑对话框。编辑完成后保存。
4. 选中表定义后，点击生成代码会弹出选择模板对话框。填写包名前缀并且选择模板后点击确定可以跳转生成代码页面
5。生成代码页面可以对代码进行查看和打包下载。
6. 控件类型管理可以维护前端控件类型，维护后在表定义中列定义选项卡的控件类型中可以选择。
7. 查询类型管理可以维护后端字段的查询类型，维护后在表定义中列定义选项卡的查询类型中可以选择。
8. 模板组管理可以进行对模板分组。
9. 模板管理是对配置的表进行代码生成的模板进行管理。可以对生成的文件名和文件内容的模板进行维护。



# 启动流程 
1. codegen目录为后端服务目录,codegen-front目录为前端页面目录
2. 本系统默认支持Mysql数据库,创建库修改好配置文件后启动后端服务则会自动初始化数据库,无需做其他配置
3. 可以通过分别启动前后端来启动系统,也可以通过前端执行npm run build:prod构建前端,然后启动后端来启动系统
